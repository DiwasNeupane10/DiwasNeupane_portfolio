## 🔥 Featured Projects

### 📝 Text Summarization Using LSA
**Technologies:** Python | Flask | MySQL | TF-IDF | SVD

<a href="https://github.com/DiwasNeupane10/Text-Summarization-Using-LSA" target="_blank">
  <img src="https://img.shields.io/badge/View%20Code-181717?style=for-the-badge&logo=github" alt="GitHub">
</a>

Extractive text summarization system using Latent Semantic Analysis with adjustable summary length.

**Key Features:**
- LSA implementation via TF-IDF & SVD for semantic understanding
- Multi-format support: PDF, TXT, DOCX, and raw text
- Evaluated using ROUGE-1, ROUGE-2, and ROUGE-N scores on DUC 2007 dataset
- User-controlled summary length adjustment

### 🎵 Audio Source Separation
**Technologies:** Python | PyTorch | Librosa | Flask

<a href="https://github.com/DiwasNeupane10/Wave-U-Net-Implementation-For-Audio-Source-Separation" target="_blank">
  <img src="https://img.shields.io/badge/View%20Code-181717?style=for-the-badge&logo=github" alt="GitHub">
</a>

Wave-U-Net model for end-to-end music source separation into drums, bass, vocals, and other instruments.

**Key Features:**
- Custom Wave-U-Net architecture built from scratch in PyTorch
- Trained on MUSDB18 dataset with preprocessing using Stempeg and Librosa
- Real-time audio separation inference pipeline
- Model checkpointing for resumable training and fine-tuning

### ⚽ Football Tactical Summary Generator
**Technologies:** HuggingFace | Unsloth | Ollama | Python

<a href="https://github.com/DiwasNeupane10/FootBall-Tactical-Summary-Generator" target="_blank">
  <img src="https://img.shields.io/badge/View%20Code-181717?style=for-the-badge&logo=github" alt="GitHub">

Fine-tuned Qwen3-4B model for generating football tactical analysis from match data.

**Key Features:**

- Instruction-based training with JSONL formatted datasets
